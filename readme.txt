Ternair Profile enables marketeers in WordPress to customize and personalize content in real-time.

== Description ==

Ternair Profile enables marketeers in WordPress to customize and personalize content in real-time, based on complete customer data profiles in Ternair (= external customer data platform).

Ternair  is a marketing automation platform for direct communication with your customers across all channels based on fully integrated customer profiles.

Ternair Profile combines the power of WordPress CMS with Ternair's customer data profiles.
see also www.ternair.com

This plugin makes use of a thirdparty service to retrieve the data that's needed to personalize the page content.
The data will only be send over a secured (https) connection.
